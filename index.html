Projects - Detailed View

Sign Language Detection

A real-time interpreter built with Python, OpenCV, MediaPipe, and a KNN classifier. Detects hand gestures via webcam and translates them into text for accessibility. The project gave me strong experience in computer vision and machine learning, particularly in applying AI to real-world accessibility challenges. Next steps include improving recognition accuracy using deep learning models (CNNs/RNNs) and adding support for full sentence detection.

Morse Code Translator

A Python-based translator that converts text ↔ Morse code. Currently supports both CLI and basic encoding/decoding logic. I used Python and Tkinter while experimenting with GUI development, adding buttons for generating dots and dashes for hands-on practice. My next step is to expand this into a full cross-platform desktop app with audio playback for Morse tones and real-time decoding from microphone input.

View on GitHub

Chrome Notes Extension

A sticky-notes Chrome extension built with HTML, CSS, JavaScript, and Chrome APIs. It allows users to create, move, resize, and persist notes across browser sessions, with data stored locally. This project taught me the fundamentals of browser extension architecture, event listeners, and persistence. Future improvements include cloud syncing between devices, customizable themes, and integrating a reminder system.

The tech stack includes React, Tailwind CSS, Node.js, and Firebase for authentication and data storage. The goal is to create something powerful, but simple enough for students.
